# goPGP

Open-source PGP encryption for mobile. https://gopgp.org

Product landing for [PGPick](https://github.com/gopgp/ios) — iOS PGP/GPG encryption app powered by [GopenPGP](https://github.com/ProtonMail/gopenpgp).

Stack: React 19 + Vite 7 + Tailwind 4.

## Develop
```
pnpm install
pnpm dev
```

## Build
```
pnpm build
```
